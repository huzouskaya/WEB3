There is the market of Costume for Party on Python in Django framework
Here you can see some spacious wearings, choose some looks for Haloween Night and keep them in mind

In realistic way you can choose sth to seek it by photo in your country

To start this project you should fork this repository and clone it by 
```
git clone
```
[use_link_in_github]
then enter in a project with
```
cd WEB3
```
[or other name you wrote down]
Then create a python virtual environment as
```
python3 -m venv venv
```
the first "venv" here is your name, you can replace it with other world

Then (on Linux)
```
source venv/bin/activate
```
[or source your_venv_name/bin/activate]

for Windows the last console line will be
```
venv\Scripts\activate.bat
```

and for the end of start
```
python3 manage.py runserver
```
congrats.
